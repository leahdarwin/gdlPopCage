# gdlPopCage

Scripts and generated figures for the paper "Parallelism, contingency, and polygenic adaptation underlies insecticide resistance in laboratory _Drosophila_ populations". 

See SCRIPTS.md for a detailed explaination of the scripts and their outputs. Short read data for pool-sequencing can be downloaded on NCBI (pending upload). Some intermediate data files referenced in the "data" directory (ex. variant calls, allele frequency files) can be downloaded from the data repository (pending upload). 

## Reproducing Figures

To reproduce figures generated by R scripts, open the root directory as an R project. All paths in R scripts are relative to the root directory and will resolve correctly when run from within the project. There are several package dependencies given at the top of each script and will need to be installed. 

Bash and Python scripts contain commands configured for a specific compute cluster (job scheduler directives, module loads) and will need to be reconfigured before running in a different environment. All paths in these scripts are also relative to the root directory.
