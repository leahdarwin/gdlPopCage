# gdlPopCage

Scripts and generated figures for the paper "Parallelism, contingency, and polygenic adaptation underlies insecticide resistance in laboratory _Drosophila_ populations". 

See SCRIPT.md for a detailed explaination of the scripts and their outputs. Short read data for pool-sequencing can be downloaded on NCBI (pending upload). Some intermediate data files referenced in the "data" directory (ex. variant calls, allele frequency files) can be downloaded from the data repository (pending upload).
